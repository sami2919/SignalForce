Why this is a Conversion-shaped artifact Salesforce two-way sync (account/contact filters, AE task creation), warehouse sync (product engagement, intent), real-time segmentation (suppression rules, dedup), three AI agents (execution, QA, optimization) with non-overlapping responsibilities, no-code orchestration of touch sequencing and triggers.

Build time on the platform: ~3 hours from blank canvas, assuming Salesforce + Snowflake already connected. The brief shape mirrors what the platform consumes — segment, sequence, agent assignments, QA rules, triggers — so this document is a config, not a doc.

What this proves: I've shipped multi-agent orchestration in production at 0–2 YOE. The shape of the work is identical: signals → segmentation → agent assignment → execution with QA → optimization. The funnel stage is different (prospecting vs MarOps), the engine is the same.
